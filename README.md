**Mkanak** is a comprehensive real estate and property booking application built with Flutter. It provides a seamless platform connecting property owners, renters, and administrators, ensuring a smooth and secure property rental experience.

📲 **[Download Mkanak on Google Play](https://play.google.com/store/apps/details?id=com.youssef.mkanak)**

---

## ✨ Key Features

Mkanak offers a tailored experience for three types of users: **Renters**, **Property Owners**, and **Admins**.

### 👤 For Renters:
* **Explore Properties:** Browse a wide variety of available properties.
* **Favorites:** Save favorite properties for quick access later.
* **Real-Time Chat:** Communicate directly with property owners.
* **Booking & Payment:** Secure in-app booking and payment gateways.
* **Order Tracking:** Monitor the status of your rental requests.

### 🏢 For Property Owners:
* **Property Management:** Easily add, edit, and manage real estate listings.
* **Order Management:** Accept or reject booking requests from renters.
* **Dashboard:** Keep track of owned properties and their availability statuses.
* **Direct Communication:** Chat with potential renters.

### 🛡️ For Administrators:
* **Approval System:** Review and approve new property listings before they go live.
* **Total Control:** Manage users, monitor orders, and oversee the entire platform ecosystem.
* **Admin Chat:** Handle customer support and system inquiries.

### ⚙️ Core Functionalities:
* **Authentication:** Secure Login and Registration system.
* **Push Notifications:** Firebase-integrated notifications to keep users updated on order statuses and new messages.
* **Location Services:** Interactive maps and location fetching for properties.
* **Responsive UI:** A beautifully crafted, customized user interface for Android and iOS.
